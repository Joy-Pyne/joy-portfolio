# Joy Pyne - Portfolio Website

Welcome to my personal portfolio website showcasing my skills, projects, education, and certifications.

## 🌟 Features

- **Responsive Design** - Mobile-friendly glassmorphism UI
- **Modern Animations** - Smooth transitions and interactive elements
- **Dark Theme** - Eye-friendly dark mode by default
- **Project Showcase** - Featured projects with GitHub links
- **Certifications** - Display of completed courses and certifications
- **Contact Form** - Easy way for visitors to get in touch

## 📋 Sections

1. **Home/Hero** - Introduction with quick skills overview
2. **About Me** - Personal background and approach to problem-solving
3. **Resume** - Tabbed view of projects, extracurricular activities, and certificates
4. **Education** - Educational background from university to high school
5. **Skills & Expertise** - Languages, frameworks, databases, and soft skills
6. **Featured Projects** - Detailed project cards with filters
7. **Certifications** - All completed courses and training
8. **Contact** - Get in touch form

## 🛠️ Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Design**: Glassmorphism, Responsive Layout
- **Icons**: Font Awesome
- **Fonts**: Google Fonts (Poppins, Inter)
- **Effects**: Particles.js, Custom Animations

## 🚀 Getting Started

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/Joy-Pyne/joy-portfolio.git
cd joy-portfolio
```

2. Install dependencies (if using Node.js):
```bash
npm install
```

3. Start the local server:
```bash
npm start
# or
node server.js
```

4. Open your browser and navigate to:
```
http://localhost:3000
```
